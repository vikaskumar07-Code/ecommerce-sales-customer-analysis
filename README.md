# ecommerce-sales-customer-analysis
Analysis of e-commerce sales and customer behavior using Python

##  Project Overview
This project analyzes e-commerce sales data to understand customer purchasing behavior, revenue trends, and product performance using Python.

##  Business Problem
The business wants to understand:
- How sales are trending over time
- Which products and categories drive revenue
- How customer behavior differs between new and repeat customers

##  Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

##  Dataset
E-commerce transaction dataset containing order, product, customer, and sales information.

##  Key Analysis
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Sales trend analysis
- Customer segmentation (new vs repeat customers)
- Product and category performance analysis

##  Key Insights
- Identified top-performing products and categories contributing most to revenue
- Found that repeat customers generate a significant portion of total sales
- Observed seasonal trends affecting sales volume

##  Recommendations
- Focus marketing efforts on high-performing products
- Improve retention strategies for repeat customers
- Optimize inventory planning based on seasonal demand

##  Outcome
This analysis helps businesses make data-driven decisions to improve sales performance and customer retention.
